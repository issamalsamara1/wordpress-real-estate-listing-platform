# WordPress Real Estate Platform

A custom WordPress platform for managing and listing real estate properties with advanced features.

## Features

- Add, edit, and delete properties
- Advanced property filtering and search
- Agent accounts management
- Image upload for properties
- Contact forms for inquiries
- Custom WordPress theme and admin dashboard
- Optimized performance and SEO-friendly

## Technologies Used

- WordPress
- PHP
- HTML5, CSS3, JavaScript
- Custom Post Types
- Advanced Custom Fields
